# Azure Sentinel SOC Lab

## Project Overview
This is a beginner-friendly Security Operations Center lab using Microsoft Azure and Microsoft Sentinel.

The goal of this project is to learn how security teams collect logs, detect suspicious activity, and investigate incidents.

## Tools Used
- Microsoft Azure
- Microsoft Sentinel
- Log Analytics Workspace
- Windows Virtual Machine
- KQL Queries
- GitHub

## Skills Learned
- Basic cloud security
- SIEM monitoring
- Log analysis
- Incident investigation
- Documentation

## Project Status
In progress.

## Lab Architecture
Windows VM → Log Analytics Workspace → Microsoft Sentinel → Security Incidents

## Next Steps
- Create Azure account
- Create Windows VM
- Connect logs to Microsoft Sentinel
- Write basic KQL queries
- Document findings
